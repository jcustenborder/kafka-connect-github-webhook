========================
GitHub WebHook Connector
========================

The GitHub WebHook connector :term:`Source Connector` that is used to receive web hook http calls caused by
interactions with GitHub.


.. warning::
    It is highly recommended that you enable :term:`SSL` with this connector. This will prevent eavesdropping of data in flight
between GitHub and the connector.
